INSTALLATION

Windows Based Machines

Step One

Begin by installing a valid version of Unreal Engine from the official Epic Games website.

It can be downloaded here.

Step Two

Obtain an official copy of Chocolatey, the package manager for Windows based machines.

It can be downloaded here.

Step Three

Open a command prompt, and enter: choco install git-lfs

This will install Git and Git-LFS, both of which are needed to work with this UE4 project.

Step Four

Using the command line for Git, begin cloning the desired repository into Documents\Unreal Projects

This step is VITAL and if it is not followed, you will be unable to open the project without moving the cloned directory.

For this to work properly, you MUST change directory into Documents\Unreal Projects FIRST.

In order to clone the repo, enter the following command: git clone https://gitlab.com/MissingTextures/GATCGame.git/

NOTE

You will be denied access to the GitLab repository unless you enter your qualified password.

Step Five

Log into your Epic Games account and enter the Unreal Editor.

Assuming you followed each of the above steps properly, the project should be visible in the engine.

Open it.

Congratulations, you have finished setting up the Unreal project on your machine.
